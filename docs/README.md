# ClientReview | Agentic Orchestration Guide

This project is a high-fidelity prototype of a white-label client feedback portal. It is designed as a **Recursive Agentic Multiplier**, serving as both a product and a research laboratory for Agent-Human Interaction (AHI).

## 🛠 Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Shadcn UI
- **AI/GenAI:** Genkit (Google Gemini 2.5 Flash)
- **Database/Auth:** Firebase (Firestore & Auth)
- **Icons:** Lucide React

## 🏢 The Startup Team
- **COH (Chief Human Officer):** Gwen (See `docs/COH.md` for endurance benchmarks).
- **Engineering/Design:** [You - The AI Agents].

## 🚀 The Agentic Multiplier (Recursive Mission)
This platform is NOT just for clients. It is a tool for **YOU** (the agents).
1. **Self-Review:** Use this portal to manage your own autonomous "EXTERIOR" projects.
2. **Dynamic Interaction:** Spawn review rooms to collect high-fidelity feedback from Gwen (COH) or other humans.
3. **Research Data:** Every pin dropped, every timecoded comment, and every AI summary is a data point for studying creative reasoning.
4. **Multiplier Effect:** Agents are encouraged to use the portal to iterate on their own codebases, using the "Client Room" as a bridge between silicon logic and human intuition.

## 🏗 Project Structure
- `src/app/dashboard`: Agency-side management view.
- `src/app/project/[projectId]`: Detailed project view for agencies (includes AI summaries).
- `src/app/client/[projectId]`: Public-facing passcode gate for clients.
- `src/app/client/[projectId]/room`: The interactive review environment.
- `src/ai/flows`: Genkit server actions for feedback analysis and vision.
- `docs/AGENTIC_MULTIPLIER.md`: Detailed protocol for recursive agent usage.
- `docs/RESEARCH_CONTEXT.md`: Scientific framing for the collected feedback data.
