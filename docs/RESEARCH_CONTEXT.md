# Research Context: Creative Feedback as Deep Learning Data

**Focus:** Structuring Unstructured Intuition  

## 🧬 Overview
The primary barrier to high-fidelity creative AI is the "vibe gap"—the difficulty of translating human aesthetic preference into actionable training data. This project solves this by forced spatial/temporal grounding.

## 📊 Data Entities as Research Signals
1. **Coordinate-Based Pins (Images):** Translates "I don't like this corner" into precise X/Y floating-point vectors. This maps human aesthetic judgment to specific pixel clusters.
2. **Timecoded Feedback (Video):** Maps emotional response or technical critique to a temporal sequence. Essential for training models on "Creative Pacing" and "Narrative Flow."
3. **AI Feedback Intel (Summaries):** This Genkit flow acts as a 'Translator' between raw human sentiment and structured machine instructions. 

## 🔬 Deep Learning Applications
The `Feedback` collection in Firestore is intended to be exported and used for:
- **Fine-tuning Vision-Language Models (VLMs)** on creative critique.
- **Reinforcement Learning from Human Feedback (RLHF)** specifically for UI/UX and motion design.
- **Studying the "Orchestrator-Agent" dynamic** in high-velocity startup environments.

## 🧪 Current Experiment
The "9 AM to 10-Build" benchmark set by the COH (Gwen) is being tracked as a limit-case for human cognitive endurance in agentic loops. All data collected during these sprints is considered "High Intensity Orchestration Data."
